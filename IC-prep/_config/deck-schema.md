# Deck Schema — `deck-outline.md`

10–15 slides. Markdown outline, machine-readable for Claude Design handoff. Claude Design produces the finished slides; this file is the spec.

## Per-slide structure

```
## Slide N — <Title>
**Key message:** <one sentence — the takeaway>
**Visual:** <chart type / image / diagram description>
**Content:**
- <bullet>
- <bullet>
**Speaker notes:** <≤ 60 words>
```

## Standard sequence (12 slides — adjust to 10–15 as needed)

| # | Slide | Source section in `memo.md` |
|---|---|---|
| 1 | Title — Company, round, ask | §2 |
| 2 | TL;DR — Why we should invest | §1 |
| 3 | Problem & Why Now | §4 |
| 4 | Product | §5 |
| 5 | Market | §6 |
| 6 | Traction | §8 |
| 7 | Business Model & Unit Economics | §7 |
| 8 | Team | §9 |
| 9 | Competition & Moat | §10 |
| 10 | Risks & Mitigations | §13 |
| 11 | Deal Terms & Return Scenarios | §§14 + 15 |
| 12 | Recommendation & Conditions | §20 |

Merge or split for complexity (e.g., separate unit-economics and GTM slides for complex deals).

## Hard rules

- One key message per slide.
- No claim in the deck that isn't in `memo.md`.
- Visuals are described in text — Claude Design generates them.
- Speaker notes ≤ 60 words.
- Total: 10–15 slides.
