# Playbook 3 — Research Synthesis

**When to use:** Dataroom populated. Convert raw inputs into structured analysis.

## Steps

1. **Synthesize findings** → `research.md`
   - Problem & timing narrative from interviews
   - Market sizing (TAM / SAM / SOM) with source attribution
   - Customer evidence (verbatim quotes where possible)
   - Honest read of traction
2. **Competitive landscape** → `competition.md` — direct + indirect, moat assessment (not founder-framed)
3. **Risk register** → `risks.md` — ≥ 5 risks with mitigations
4. **Portfolio conflict check** → `conflicts.md`
   - Dereference operating fund `portfolio.md` (see root `CLAUDE.md`)
   - Check each portfolio company vs. target — direct + adjacent
   - Explicit Pass / Conflict result
5. **Syndication plan** → `syndication.md`
   - ≥ 3 candidate co-investors via `funds/` brief scans
   - Filter each through `funds/playbooks/syndicating-a-deal.md`
6. **Update `notes.md` open questions** — what's still unknown

## Exit criterion

All items in `_config/review-rubric.md` "After stage 3" met.

## Reroute back

- Evidence gap (claim with no source) → stage 2
- Market sizing not defensible → stage 2 (more research) or revise downward
