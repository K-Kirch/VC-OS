# Playbook 6 — Final Package

**When to use:** Review gate passed. Assemble for IC.

## Steps

1. **Final pass on `memo.md`** — typos, formatting, numbers double-checked
2. **Deck production** — hand `deck-outline.md` to Claude Design; place finished deck under `dataroom/deck/ic-deck-<YYYY-MM-DD>.<ext>`
3. **Appendix bundle** — gather appendix items per `appendix.md` into a single IC-facing folder or PDF
4. **Distribute** — per IC's standard distribution channel
5. **Update `notes.md`** — file status all = Yes, last_updated today
6. **Update `deal.md`** — `current_stage` = `Decided` (after IC)

## Post-IC

1. **Capture vote** in `recommendation.md`
2. **Log decision** in `review-log.md`
3. **Update `deal.md` status** — `Active` → `Closed` / `Passed` / `Killed`
4. **If invested:** create or link to portfolio company tracking (out of scope for IC-prep)

## Exit criterion

Deal decided. Artifacts archived. Memory captured.
