# Playbook 4c — Appendix Assembly

**When to use:** Memo draft exists. Curate supporting material for IC.

## Steps

1. **Open `appendix.md`** — list every item that travels with the IC package
2. **Inclusion criteria:**
   - Referenced by a `memo.md` claim
   - Likely to be requested by IC (use prior IC questions as a guide)
   - Needed to defend `recommendation.md` under questioning
3. **For each item:** name, source location in `dataroom/` or working file, memo reference
4. **Trim aggressively** — if it doesn't earn its place, leave it in `dataroom/` and out of the appendix

## Hard rules

- Appendix is a *curated subset*, not the whole dataroom
- Each appendix item maps to a memo section reference

## Exit criterion

`appendix.md` complete.
