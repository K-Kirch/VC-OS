# Playbook 2 — Dataroom Assembly

**When to use:** Deal has cleared intake. Gather everything needed to evaluate.

## Steps

1. **Request founder materials** — deck (latest), financial model, cap table, key contracts. Place in `dataroom/deck/`, `dataroom/financials/`, `dataroom/legal/`
2. **Founder call(s)** — write up in `dataroom/interviews/YYYY-MM-DD-<founder>-founder.md`
3. **Customer interviews** — target ≥ 3. One file per call: `dataroom/interviews/YYYY-MM-DD-<name>-customer.md`
4. **References** — ≥ 2 backchannels on the founding team. `dataroom/references/`
5. **Product observations** — demo notes, technical DD → `dataroom/product/`
6. **Market materials** — industry reports, analyst notes → `dataroom/market/`
7. **Update `dataroom/INDEX.md`** — inventory current as of today

## Exit criterion

All items in `_config/review-rubric.md` "After stage 2" met.

## Reroute back

- Founder unresponsive on materials → escalate before proceeding
- Reference calls unavailable → document why; may proceed with risk flag
